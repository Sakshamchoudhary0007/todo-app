# React To-Do App

This is a simple To-Do application built with ReactJS that allows users to add, view, delete, and edit tasks.

## Features

- Add tasks
- View tasks
- Delete tasks
- Edit tasks
- Optional: Mark tasks as completed
- Optional: Persistent storage using local storage
